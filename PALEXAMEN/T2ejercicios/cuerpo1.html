En programación, los tipos de datos son una parte fundamental que define el tipo de valores que una variable puede contener y las operaciones que se pueden realizar con esos valores. Aquí tienes información sobre algunos tipos de datos comunes: Enteros (integers): Representan números enteros sin parte fraccionaria. Pueden ser tanto positivos como negativos. Ejemplos en varios lenguajes de programación: int en Python, int en Java, int en C++. Punto flotante (floating-point): Representan números con parte fraccionaria. Pueden ser positivos o negativos. Ejemplos: float en Python, double en Java, float o double en C++. Cadenas de caracteres (strings): Representan secuencias de caracteres. Se utilizan para manipular texto. Ejemplos: str en Python, String en Java, string en C++. Booleanos: Representan valores de verdad, es decir, verdadero (true) o falso (false). Utilizados en expresiones lógicas y de control de flujo. Ejemplos: bool en Python, boolean en Java, bool en C++. Listas (arrays): Almacenan colecciones ordenadas de elementos del mismo tipo. Se accede a los elementos mediante un índice. Ejemplos: list en Python, array en Java, vector en C++. Tuplas: Similar a las listas, pero son inmutables (no se pueden modificar después de su creación). Los elementos pueden ser de diferentes tipos. Ejemplos: tuple en Python. Diccionarios (mapas, hashmaps): Almacenan pares clave-valor, permitiendo acceder a los valores a través de sus claves. No tienen un orden específico. Ejemplos: dict en Python, Map en Java, unordered_map en C++. Sets: Almacenan elementos únicos en un orden no especificado. Utilizados para operaciones de conjuntos como unión, intersección, etc. Ejemplos: set en Python, Set en Java, unordered_set en C++. Caracteres (char): Representa un solo carácter. Se utiliza para almacenar caracteres individuales. Ejemplos: char en C++.
